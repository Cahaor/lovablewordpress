Cómo instalar el tema

  1. Sube el ZIP del proyecto compilado (con carpeta dist/ o build/)
  2. Escribe el nombre del tema y pulsa Convertir
  3. Descarga el ZIP generado
  4. En WordPress: Apariencia → Temas → Añadir nuevo → Subir tema
  5. Activa el tema. Listo — tu web aparece como portada
